## Project Background
  디지털 기술이 빠르게 발전함에 따라 기업과 기관들은 정보 전달의 효율성을 극대화해야 하는 요구에 직면하고 있습니다. 기존의 리플렛, 포스터, 직원 안내 등 아날로그 방식은 정보 수정과 유지관리가 까다롭고, 실시간 전달력이 부족하다는 근본적인 한계가 있습니다. 
  이에 본 연구는 터치 인터페이스와 MSA(Microservice Architecture, 마이크로서비스 아키텍처)를 결합한 사용자 맞춤형 디지털 정보 시스템 개발을 목표로 삼았습니다. 터치패널 기반의 정보 시스템은 실시간으로 콘텐츠를 업데이트하며 멀티미디어 활용, 개별 맞춤형 서비스 제공이 가능해 조직 내외 정보 전달 방식을 혁신할 수 있습니다. 또한, 정보 접근성 향상, 브랜드 이미지 강화, 운영 효율 증대 등 다양한 효과를 기대할 수 있습니다.

## System Model
본 시스템은 MSA 구조에 기반해 여러 개의 독립 마이크로서비스로 구성되었습니다. 각 서비스는 공지사항, 이벤트 안내, 성과 공유 등 기능별로 분리되어 독립적으로 개발, 배포 및 관리가 이루어집니다. 백엔드는 TypeScript 및 NestJS, PostgreSQL이 활용되어 신뢰성과 확장성을 확보했고, 프론트엔드는 React와 Vite로 구현되어 터치패널 환경에서 빠르고 직관적인 UX를 제공합니다. 데이터는 각 서비스별 자체 데이터베이스에 분산 저장되며, 서비스 간 통신은 RESTful API를 통해 이뤄집니다. API 연동을 통해 ERP, 인트라넷 등 기존 IT 인프라와의 통합도 용이하며, 모든 서비스는 중앙 모니터링과 자동화 배포(CI/CD) 시스템 하에 운영됩니다.

## Numerical Results
  실제 환경에 시스템을 배포한 결과, 정보 콘텐츠 수정 및 공지사항 반영 시간이 기존 아날로그(포스터 등) 방식 대비 90% 이상 단축되었습니다. 회의실 예약의 경우, 수동/중복 관리 문제가 해소되어 예약 안정성이 크게 향상되었고, 관리자 페이지 통합 관리 덕분에 각 부서 간 정보 전달 실수가 현저하게 줄었습니다. 사용자(직원, 방문객)의 터치 UI 활용률 증대 및 고유 방문 이력 분석을 통해 학과 및 기업의 홍보 효과가 크게 상승했습니다.

## Conclusion
  MSA 구조와 터치 기반 인터페이스를 기업 및 학과 홍보 시스템에 적용함으로써, 서비스의 기능별 확장성, 유지보수 편의성을 실현하고 실제 현장에 배포해 유의미한 효과를 거둘 수 있었습니다. 
  정적 시스템의 한계를 넘어서 즉각적 정보 업데이트, 멀티미디어 활용, 실시간 데이터 관리가 가능해지며, 운영 효율과 사용자 만족도도 크게 향상되었습니다. 프로젝트 진행 과정에서는 실제 기업 요구와 피드백을 적극적으로 반영하여 세부 설계를 반복 개선했습니다.

## Project Outcome
  학과 키오스크 및 기업 정보 안내용 통합 플랫폼의 구축·운영을 통해 실제 현장에서 가시적 성과(정보 전달, 예약 안정성, 홍보 효과, 운영비 절감 등)를 거둘 수 있었습니다. 
  배포 환경도 오픈소스·자동화 시스템으로 설계되어, 다른 기업이나 기관에서도 손쉽게 확장·응용할 수 있도록 준비되었습니다. 
  또한 CI/CD 및 Docker, Raspberry Pi 기반 운영을 통해 지속적 배포와 관리가 자동화되어 미래 기능 확장과 유지보수에도 중요한 기술적 기반을 확보하였습니다.
